Tools
=====

Tools available in gptme.

  - `Shell`_

  - `Tmux`_
  - `Subagent`_

- Files

  - `Read`_
  - `Save`_
  - `Patch`_

- Network

  - `Browser`_

- Vision

  - `Screenshot`_
  - `Vision`_
  - `Computer`_

- Chat management

  - `Chats`_

- Context management

  - `RAG`_
